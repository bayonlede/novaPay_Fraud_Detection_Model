üîç Understanding SHAP Analysis

SHAP (SHapley Additive exPlanations) is a game-theoretic approach that explains how each feature contributed to the model's prediction. It answers the question: "Why did the model make this decision?"

üìä What is SHAP?

Traditional machine learning models are often called "black boxes" because they don't naturally explain their decisions. SHAP transforms our LightGBM model into a transparent, explainable system by calculating how much each of the 22 features contributed to the fraud probability score.
